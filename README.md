# めくり

## 📖 サービス概要
**「未来をめくる、今日を破る。」**

「めくり」は、目標の日までを「紙を破る」というアクションでカウントダウンする、新感覚の日めくりカレンダーWebアプリケーションです。

子供の頃に感じた「あと何日？」というワクワク感を、デジタル時代に合わせて再構築。物理的な日めくりカレンダーのような「めくる心地よさ」をWeb上で再現しつつ、LINEやプッシュ通知（PWA）によるリマインドで、目標当日までの継続をしっかりサポートします。

## 💡 サービスへの思い・開発背景
子供の頃、夏休みや冬休みまで「あと何日」と、カレンダーと毎日睨めっこしていた記憶はないでしょうか。大人になった今も、「楽しみにしている旅行」や「好きなアーティストのライブ」など、待ち遠しい日はたくさんあります。

しかし、日々の忙しさの中でそのワクワク感を忘れてしまいがちです。

* **「未来を待つ童心」を思い出してほしい**
* **デジタルな画面の中で「紙を破る」アナログな手触りを楽しんでほしい**

そんな思いから、ただ数字が減るだけでなく、毎日手でめくる**「感触」と「過程」**を大切にした「めくり」を開発しようと考えました。

## 📣 ユーザー獲得

### SNSシェアによるバイラルループ
「めくり」の最大の価値である「ワクワク感」や「達成感」は、誰かに共有したい感情です。
特に「推し活」や「受験勉強」の文脈では、進捗をSNSで報告する文化が根付いています。

* **動的OGPの実装**
  * SNSシェア時に、動的OGPを生成する仕組みを導入予定。
  * 見た人が「自分もやりたい」と感じる導線を作ります。
* **モーメントを逃さないUI**
    * カレンダーを「めくった直後（達成感がある瞬間）」にシェアボタンを配置し、自然な投稿を促します。

## 🌟 サービスの特徴・差別化ポイント

従来のカレンダーアプリやカウントダウンアプリは「記録」や「通知」に特化しており、情緒的な価値が不足していました。「めくり」は以下の点で差別化を図っています。

### 1. 「体験」としてのカウントダウン
既存のアプリが「今日の日付を確認する」ものであるのに対し、本サービスは**「目標に向かって紙を破り捨てる」**というアクション自体に価値を置いています。

### 2. プロセスを愛でる「日記モード」
未来を待つだけでなく、めくったカレンダーの裏側に「その日の記録」を残すことで、当日を迎えた時に「積み上げてきた過去」も同時に振り返ることができる設計です。

## 🛠 機能一覧

### ✨ MVPリリース機能（基本機能）
* **ユーザー認証**
  * LINEログイン（「LINEで始める」ボタン1つで登録・ログイン）
  * Googleログイン（PWA利用を想定）
* **日めくりカレンダー管理**
  * カレンダー作成（タイトル、目標日設定）
  * カード形式での一覧表示
  * ステータス管理（挑戦中 / 達成済み）
  * 実績表示（これまでにめくった枚数の集計）
  * 達成したらSNSへシェア
* **日めくりアクション (Core Feature)**
  * 残日数の計算と表示
  * **インタラクション:** 日めくりカレンダーのアニメーション
  * **制約:** 1カレンダーにつき1日1回のみ（「明日」はめくれない）
* **通知・リマインド**
  * 未実施ユーザーへの自動通知（毎日定時バッチ処理）
  * LINE通知 / プッシュ通知（Web Push）

### 🚀 ロードマップ（今後の実装予定）
* **習慣化・記録機能の強化**
  * **タスクモード:** 翌日のタスクを宣言しないとめくれない機能
  * **日記モード:** 今日の振り返り（200文字程度）を入力してめくる機能
  * **履歴の可視化:** 破ったカレンダー（過去の日々）を一覧で振り返る
* **PWAアップデート**
  * オフライン対応（Service Workerによる画像キャッシュ）
* **ソーシャル機能**
  * **パートナー共有:** 招待リンクによるカレンダー共有
  * **共同めくり:** 誰か1人がめくれば完了する「交換日記」のような体験
