# めくり

## 📖 サービス概要
**「未来をめくる、今日を破る。」**

「めくり」は、目標の日までを「紙を破る」というアクションでカウントダウンする、新感覚の日めくりカレンダーWebアプリケーションです。

子供の頃に感じた「あと何日？」というワクワク感を、デジタル時代に合わせて再構築。物理的な日めくりカレンダーのような「めくる心地よさ」をWeb上で再現しつつ、LINEやプッシュ通知（PWA）によるリマインドで、目標当日までの継続をしっかりサポートします。

## 💡 サービスへの思い・開発背景
子供の頃、夏休みや冬休みまで「あと何日」と、カレンダーと毎日睨めっこしていた記憶はないでしょうか。

私自身、旅行が大好きで、よく数ヶ月先の航空券を予約してはその日を待ち望んでいます。予約した瞬間は「あと3ヶ月！」と胸が高鳴りますが、日々の忙しさに追われていると、そのワクワク感はいつの間にか日常のタスクに埋もれてしまいがちです。
気づけば出発の前日になり、「あれ？もう明日だっけ？」と、一番楽しいはずの準備期間（待ち時間）を十分に味わえずにもったいないと感じることがありました。

「楽しみにしている旅行」や「好きなアーティストのライブ」など、大人になった今も待ち遠しい日はたくさんあります。

* **「未来を待つ童心」を思い出してほしい**
* **デジタルな画面の中で「紙を破る」アナログな手触りを楽しんでほしい**

そんな思いから、ただ数字が減るだけでなく、毎日手でめくる**「感触」と「過程」**を大切にした「めくり」を開発しようと考えました。

## 📣 ユーザー獲得

### SNSシェアによるバイラルループ
「めくり」の最大の価値である「ワクワク感」や「達成感」は、誰かに共有したい感情です。
特に「推し活」や「受験勉強」の文脈では、進捗をSNSで報告する文化が根付いています。

* **動的OGPの実装**
  * SNSシェア時に、動的OGPを生成する仕組みを導入予定。
  * 見た人が「自分もやりたい」と感じる導線を作ります。
* **モーメントを逃さないUI**
    * カレンダーを「めくった直後（達成感がある瞬間）」にシェアボタンを配置し、自然な投稿を促します。

## 🌟 サービスの特徴・差別化ポイント

世の中には多くのカレンダーアプリやカウントダウンアプリが存在しますが、それらの多くは「効率的なスケジュール管理」や「単なる残日数の表示」に留まっています。「めくり」は以下の点で明確な差別化を図っています。

### 1. 「確認」ではなく「儀式」としての体験
既存アプリは自動的に残日数が減っていきますが、「めくり」は**ユーザー自身の手で紙を破らないと日付が進みません**。
この「あえて手間をかける」アナログなアクションを取り入れることで、ただ時間が過ぎるのを待つのではなく、自らの手で目標を手繰り寄せる「儀式」のような体験を提供します。

### 2. 「待ち時間」を「コンテンツ」に変える
旅行やライブまでの期間は、通常「早く過ぎてほしい時間」になりがちです。
本サービスは、日めくりのアクション自体に心地よさ（アニメーションや音）を持たせることで、**待っている期間そのものを楽しむコンテンツ**へと昇華させます。

### 3. プロセス（過去）の可視化
単に未来をカウントダウンするだけでなく、日記機能（ロードマップ）によって「めくった裏側」に記録を残せる設計です。
これにより、目標当日に到達した際、**「ただその日が来た」だけでなく「これだけ積み上げてきた」というストーリー**を感じることができます。

## 🛠 機能一覧

### ✨ MVPリリース機能（基本機能）
* **ユーザー認証**
  * LINEログイン（「LINEで始める」ボタン1つで登録・ログイン）
  * Googleログイン（PWA利用を想定）
* **日めくりカレンダー管理**
  * カレンダー作成（タイトル、目標日設定）
  * カード形式での一覧表示
  * ステータス管理（挑戦中 / 達成済み）
  * 実績表示（これまでにめくった枚数の集計）
  * 達成したらSNSへシェア
* **日めくりアクション (Core Feature)**
  * 残日数の計算と表示
  * **インタラクション:** 日めくりカレンダーのアニメーション
  * **制約:** 1カレンダーにつき1日1回のみ（「明日」はめくれない）
* **通知・リマインド**
  * 未実施ユーザーへの自動通知（毎日定時バッチ処理）
  * LINE通知 / プッシュ通知（Web Push）

### 🚀 ロードマップ（今後の実装予定）
* **習慣化・記録機能の強化**
  * **タスクモード:** 翌日のタスクを宣言しないとめくれない機能
  * **日記モード:** 今日の振り返り（200文字程度）を入力してめくる機能
  * **履歴の可視化:** 破ったカレンダー（過去の日々）を一覧で振り返る
* **PWAアップデート**
  * オフライン対応（Service Workerによる画像キャッシュ）
* **ソーシャル機能**
  * **パートナー共有:** 招待リンクによるカレンダー共有
  * **共同めくり:** 誰か1人がめくれば完了する「交換日記」のような体験
